# Quotify

## Metadata
- **Category**: Social
- **Created**: 2025-11-27
- **Updated**: 2025-11-27

## Description
Create beautiful quote images with elegant typography and gradient backgrounds. Export your quotes as high-quality images perfect for social media sharing.

## Features
- Live preview of your quote card
- 6 beautiful gradient styles
- Clean, elegant typography
- Author attribution
- Modern framed card design
- Export as high-resolution PNG
- Mobile-responsive design

## Usage
1. Enter your quote text
2. Optionally add an author name
3. Choose a gradient style
4. Click "Download Image" to save as PNG
