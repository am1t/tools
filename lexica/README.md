# Lexica

## Metadata
- **Category**: Learn
- **Created**: 2025-11-27
- **Updated**: 2025-11-27

## Description
An interactive flashcard app with 108 curated English words to help non-native speakers write with more precision and nuance. Each session randomly selects 5-10 cards to practice.

## Features
- 108 carefully selected words for better writing
- Each word includes definition, synonyms, usage tips, and examples
- Click to flip cards and reveal full details
- Random selection of 5-10 cards per session
- Progress indicator and easy navigation
- Clean, minimal design

## Usage
Open the tool and click on any card to flip it. Use the Previous/Next buttons to navigate through your session's cards. Refresh the page to get a new random set of words to study.
