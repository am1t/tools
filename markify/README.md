## Description
Convert HTML content to clean Markdown format with automatic HTML entity decoding and whitespace cleanup.

## Features
- Paste HTML and get clean Markdown output
- Handles common HTML elements (headers, links, lists, tables, code blocks, etc.)
- Decodes HTML entities (&amp;, &lt;, &gt;, &nbsp;, &quot;, etc.)
- Cleans up excessive whitespace
- One-click copy to clipboard
- Preserves code blocks and inline code formatting

## Usage
1. Paste your HTML content into the input area
2. Markdown automatically appears as you type
3. Click "Copy Markdown" to copy to clipboard
4. Use the "Clear" button to start over

Perfect for converting web content, documentation, or any HTML snippets to Markdown format.
