## Description
A simple, clean counter tool with increment, decrement, and reset functionality. Perfect for counting anything - people, items, repetitions, or keeping score.

## Features
- Increment and decrement counter
- Reset to zero
- Keyboard shortcuts (Arrow Up/+, Arrow Down/-, R for reset)
- Persistent state using localStorage (your count is saved between sessions)
- Clean, modern interface
- Works offline

## Usage
Access at `/counter/` or open `index.html` directly in your browser. Use the buttons or keyboard shortcuts to interact with the counter. Your count is automatically saved and will persist when you return.
