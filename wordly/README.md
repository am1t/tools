# Wordly

## Metadata
- **Category**: Text
- **Created**: 2025-11-26
- **Updated**: 2025-11-26

## Description
A simple tool to count characters, words, and lines in any text. Automatically strips HTML and Markdown formatting. Perfect for writers, students, and anyone who needs to track text statistics.

## Features
- Real-time character counting (with and without spaces)
- Word count
- Line count
- Automatic HTML and Markdown stripping
- Clean, intuitive interface
- Instant updates as you type

## Usage
Simply paste or type your text into the text area, and the tool will automatically strip HTML and Markdown formatting, then count characters, words, and lines in real-time.
